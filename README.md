# Discrete-OP-Amp-
Simple discrete OP amp simulations and schematic diagram
This repo holds LTSPICE simulations for both the open loop gain and the buffer configuration of a simple discrete op amp this design is made with CMOS ALD1101 and ALD1102 transistor pair but it can be made with common NPN and PNP transistor without modifications to the schematic of the simulation diagram

For this simple design the open loop gain is 110dB max and the GBP is 10MHz
